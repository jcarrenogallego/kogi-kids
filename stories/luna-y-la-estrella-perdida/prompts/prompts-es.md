## 🚀 Cómo Usar (Flujo de Trabajo Web UI)

**IMPORTANTE**: Estos prompts están diseñados para la **Web UI de MidJourney**, NO para Discord.

### Paso a Paso para Cada Prompt:

1. **Entrá a la Web UI de MidJourney** (https://midjourney.com/imagine)
2. **Subí la Imagen de Referencia del Personaje** (click "Image Prompts" button):
   - Para prompts con Luna: Subir `moodboard/luna.png`
   - Para prompts con Estrellita: Subir `moodboard/estrellita.png`
   - Para prompts con Oliver: Subir `moodboard/oliver.png`
   - Para prompts con múltiples personajes: Subir 2-3 imágenes según necesites
3. **Copiá el prompt** del bloque de código de abajo (texto en INGLÉS - MidJourney funciona mejor con prompts en inglés)
4. **Pegá en el campo de prompt** (without `/imagine`)
5. **Generá** y elegí la mejor variación
6. **Descargá** y guardá en la carpeta `renders/` con el nombre apropiado
